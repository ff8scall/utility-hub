## 🌟 Features

### 📐 단위 변환 (Unit Converters)
- **길이 변환기** - 미터, 피트, 인치 등 단위 변환
- **무게 변환기** - 킬로그램, 파운드 등 단위 변환
- **환율 계산기** - 주요 통화 환율 변환 (정적 환율)
- **온도 변환기** - 섭씨, 화씨, 켈빈 온도 변환
- **면적 변환기** - 제곱미터, 평, 에이커 등 면적 단위 변환
- **부피 변환기** - 리터, 갤런, 세제곱미터 등 부피 단위 변환
- **속도 변환기** - km/h, mph, m/s, 노트 등 속도 단위 변환

### 💪 건강 (Health)
- **BMI 계산기** - 체질량 지수 계산 및 건강 상태 확인
- **BMR 계산기** - 기초대사량 및 하루 권장 칼로리 계산
- **청력 나이 테스트** - 고주파수 청력 테스트
- **동체 시력 테스트** - 움직이는 물체 식별 테스트

### 🎮 게임 (Games)
- **반응속도 테스트** - 나의 순발력 측정

- **타자 속도 테스트** - 한글/영어 타자 속도(WPM)와 정확도 측정
- **1 to 50** - 1부터 50까지 빠르게 터치하기
- **CPS 테스트** - 초당 마우스 클릭 속도 측정
- **에임 트레이너** - 마우스 정확도 및 반응속도 훈련
- **숫자 기억하기** - 순간 기억력 테스트 (침팬지 테스트)
- **숫자 야구** - 숫자와 위치를 맞추는 추리 게임
- **지뢰찾기** - 지뢰를 피해 모든 칸을 여는 퍼즐
- **돌림판 돌리기** - 랜덤 추첨을 위한 룰렛 게임
- **사다리 타기** - 내기할 때 좋은 사다리 게임
- **스네이크 게임** - 먹이를 먹고 길어지는 클래식 뱀 게임
- **스도쿠** - 클래식 9x9 로직 퍼즐 게임
- **탑 쌓기** - 타이밍에 맞춰 블록을 높이 쌓는 게임
- **타자 디펜스 게임** - 떨어지는 단어 파괴 게임
- **절대음감 테스트** - 음정 맞추기 게임
- **절대 시간 감각 테스트** - 10.00초에 정확히 멈추는 도전
- **세계 수도 퀴즈** - 국기를 보고 수도 맞추기
- **행맨 게임** - 영어 단어 추리 게임

### 💰 생활/금융 (Finance & Life)
- **대출금 계산기** - 원리금 균등상환 방식의 대출 이자 계산
- **연봉 실수령액** - 4대 보험과 세금을 제외한 실수령액 계산
- **복리 계산기** - 복리 이자 및 투자 수익 계산
- **근무시간 계산기** - 근무 시간 및 급여 계산
- **나이 계산기** - 만 나이, 연 나이 계산
- **날짜 계산기** - 날짜 차이, D-day 계산
- **할인 계산기** - 할인율 계산 및 역계산
- **퍼센트 계산기** - 백분율, 증감률, 비율 계산
- **분수 계산기** - 분수 사칙연산 및 기약분수 변환

### 📝 텍스트 도구 (Text Tools)
- **글자 수 세기** - 글자, 단어, 문장 수 계산
- **유니코드 변환기** - 텍스트 ↔ 유니코드 변환
- **문자열 변환기** - 대소문자, 카멜케이스 등 변환
- **Base64 도구** - Base64 인코딩/디코딩
- **HTML 인코더** - HTML 엔티티 인코딩/디코딩
- **모스 부호 번역기** - 텍스트와 모스 부호 변환
- **한국어 맞춤법 능력 고사** - 알쏭달쏭한 맞춤법 퀴즈
- **어휘력 테스트** - 사자성어 및 문해력 테스트

### 💻 개발자 도구 (Developer Tools)
- **진법 변환기** - 2진수, 8진수, 10진수, 16진수 변환
- **JSON 포맷터** - JSON 데이터 정리 및 검증
- **마크다운 에디터** - 실시간 미리보기 마크다운 편집기
- **HTML 포맷터** - HTML 코드 정리 및 미리보기
- **코드 비교** - 두 텍스트/코드의 차이점 비교
- **웹 에디터** - HTML/CSS/JS 실시간 편집 및 미리보기
- **해시 생성기** - MD5, SHA-1, SHA-256 등
- **UUID 생성기** - UUID v4 생성
- **아스키아트** - 텍스트를 아스키 아트로 변환
- **아스키 코드표** - ASCII 코드 참조표
- **특수문자표** - 자주 쓰는 특수문자 모음
- **URL 인코더/디코더** - URL 인코딩 및 디코딩
- **JWT 디코더** - JWT 토큰 디코딩 및 확인
- **정규식 테스터** - 정규표현식 테스트 및 검증
- **CRON 표현식 생성기** - CRON 스케줄링 표현식 생성
- **CSV ↔ JSON 변환기** - CSV와 JSON 상호 변환

### 🔧 유틸리티 (Utilities)
- **QR코드 생성기** - WiFi, URL, 명함 QR코드 생성
- **바코드 생성기** - 숫자를 바코드로 변환
- **비밀번호 생성기** - 안전한 랜덤 비밀번호 생성
- **컬러 피커** - HEX, RGB, HSL 색상 선택
- **색상 추출기** - 이미지에서 컬러 팔레트 추출
- **이미지 Base64** - 이미지를 Base64로 인코딩
- **IP 주소 확인** - 현재 IP 주소 확인
- **타이머/스톱워치** - 온라인 타이머 및 스톱워치
- **포모도로 타이머** - 25분 집중/5분 휴식 타이머
- **체크리스트** - 간단한 할 일 목록
- **손전등** - 화면을 밝게 (모바일용)
- **이미지 리사이저** - 이미지 크기 조정 및 압축
- **유튜브 썸네일** - 유튜브 영상 썸네일 다운로드
- **세계 시계** - 전 세계 주요 도시 현재 시간
- **암호화/복호화** - AES 알고리즘 기반 텍스트 암호화
- **주사위 굴리기** - D4, D6, D8, D10, D12, D20, D100 주사위

### 🎭 운세/재미 (Fun & Fortune)
- **혈액형 성격** - 혈액형별 성격 분석
- **MBTI 테스트** - 간단한 MBTI 성격 유형 검사
- **무료 만세력** - 생년월일로 사주팔자 확인
- **띠별 운세** - 12띠 오늘의 운세
- **별자리 운세** - 12별자리 오늘의 운세
- **꿈해몽** - 꿈 내용으로 해몽 검색
- **성명학** - 이름 획수로 운세 분석
- **바이오리듬** - 신체/감성/지성 리듬 확인
- **탄생석/탄생화** - 월별 탄생석과 탄생화 확인
- **타로 카드** - 오늘의 타로 운세
- **로또 번호 생성기** - 행운의 로또 번호 자동 생성

## 🚀 Tech Stack

- **Frontend**: React 19.2.0
- **Build Tool**: Vite 7.2.2
- **Styling**: TailwindCSS 3.4.17
- **Routing**: React Router 7.1.1 (BrowserRouter)
- **Icons**: Lucide React
- **Deployment**: GitHub Pages
- **CI/CD**: GitHub Actions

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/ff8scall/utility-hub.git

# Navigate to project directory
cd utility-hub

# Install dependencies
npm install

# Start development server
npm run dev
```

## 🔨 Development

```bash
# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint code
npm run lint
```

## 🌐 Deployment

This project is automatically deployed to GitHub Pages using GitHub Actions.

**Live URL**: https://ff8scall.github.io/utility-hub/

Every push to the `main` branch triggers an automatic build and deployment.
