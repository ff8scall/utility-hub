# Saju 2.0 Update (사주팔자 2.0 업데이트)

## 📅 Release Date: 2025-11-20

## 🎯 Major Updates

### 1. UI/UX 대폭 개선 - 탭 인터페이스 도입
- **기존**: 단일 페이지 스크롤 방식으로 모든 정보 표시
- **개선**: 5개 탭으로 분리하여 정보 과부하 방지
  - 기본 분석: 사주팔자, 오행, 성격 분석
  - 인생 흐름: 대운, 세운, 오늘의 운세
  - 신살: 신살 분석 및 설명
  - 궁합: 파트너와의 궁합 분석
  - 달력: 월별 운세 달력

### 2. 신살(神殺) 분석 확장
- **기존**: 도화살, 역마살, 화개살 (3종)
- **추가**: 
  - 귀문관살 (鬼門關殺): 직관력, 영감
  - 백호대살 (白虎大殺): 강인함, 리더십
  - 현침살 (懸針殺): 정밀함, 전문 기술
  - 천을귀인 (天乙貴人): 귀인의 도움, 행운
  - 문창귀인 (文昌貴人): 학문, 시험운
- **총 8가지** 신살 자동 감지 및 상세 해석

### 3. 궁합(相性) 분석 기능 추가 ⭐ NEW
- 파트너의 생년월일 입력
- **천간합(天干合)** 체크: 일간의 조화 분석
- **오행 균형** 분석: 서로의 부족한 오행을 채워주는지 확인
- **궁합 점수** 계산: 0-100점 척도
- 상세 해석 및 조언 제공

### 4. 운세 달력 기능 추가 ⭐ NEW
- 월별 달력 뷰 (이전/다음 달 이동)
- 매일의 **일진(日辰)** 자동 계산 및 표시
- 나의 일간과의 관계를 통한 **일별 운세** 분석
- 좋은 날 / 주의할 날 시각적 표시
- 날짜 클릭 시 상세 운세 확인

### 5. 공유 기능 추가 ⭐ NEW
- 클릭 한 번으로 결과 복사
- 사주팔자, 일간, 신살 등 핵심 정보 포맷팅
- 카카오톡, 메신저 등으로 쉽게 공유 가능

### 6. 코드 리팩토링
- 대용량 데이터 분리: `src/data/SajuData.js` 생성
- 컴포넌트 모듈화: `src/pages/saju_components/` 디렉토리
  - `SajuBasic.jsx`: 기본 분석
  - `SajuFlow.jsx`: 인생 흐름
  - `SajuShinsal.jsx`: 신살 분석
  - `SajuCompatibility.jsx`: 궁합 분석
  - `SajuCalendar.jsx`: 운세 달력
- 유지보수성 및 확장성 대폭 향상

## 🔧 Technical Improvements

### Performance
- 컴포넌트 분리로 초기 로딩 속도 개선
- 탭 기반 렌더링으로 메모리 효율성 증가

### Code Quality
- 데이터와 로직 분리 (Separation of Concerns)
- 재사용 가능한 컴포넌트 구조
- Props 전달 최적화

### User Experience
- 직관적인 탭 네비게이션
- 모바일 반응형 디자인 유지
- 애니메이션 효과로 부드러운 전환
